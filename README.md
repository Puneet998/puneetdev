#  Personal Portfolio Website

A modern, fully responsive **Personal Portfolio Website** built using **React + TypeScript** for the frontend and **Express.js** for the backend.  
This portfolio showcases my **skills, projects, certificates, and contact details**, along with a working **email/contact system**.

---

##  Features

-  **About Me** – Complete personal and professional introduction  
-  **Skills Section** – Technologies and tools I work with  
-  **Projects** – Real-world projects with details  
-  **Certificates** – Certifications and achievements  
-  **Contact Form** – Email integration using Express.js  
-  **Fully Responsive** – Works smoothly on all devices  
-  **Fast & Optimized** – Built with modern technologies  

---

##  Tech Stack

### Frontend
- React.js  
- TypeScript  
- HTML5  
- CSS3  

### Backend
- Node.js
